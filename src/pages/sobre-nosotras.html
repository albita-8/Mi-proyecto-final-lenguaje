🎬 Nuestro Proyecto y Base de Datos

La razón principal por la que decidimos diseñar y desarrollar esta base de datos es porque nos parece un tema muy interesante. Creemos que el universo Disney ofrece una oportunidad fantástica para construir una plataforma web muy creativa.

Es un mundo que cuenta con un potencial enorme y muchísimo contenido por explotar, desde películas memorables y personajes icónicos hasta las canciones que marcan nuestra infancia.

💻 Tecnologías Utilizadas

Para hacer posible la magia de nuestra API y la interfaz de esta página, estamos trabajando con el siguiente stack tecnológico:

Frontend (Interfaz de Usuario)

  • HTML5: Estructura y semántica de la web.
  • CSS3: Diseño, estilos y animaciones.
  • JavaScript: Interactividad y consumo de la API.

Backend & Servidor (The API of Wonderland)

  • Node.js: Entorno de ejecución para nuestro servidor.
  • Express.js: Framework para la creación y gestión de rutas de la API.
  • Multer: Middleware para la subida y gestión de archivos (como imágenes o audios).
  • CORS: Middleware para permitir y gestionar de forma segura las peticiones cruzadas entre nuestro frontend y backend.

Base de Datos

  • MySQL: Sistema de gestión de bases de datos relacional para almacenar nuestros personajes, películas y canciones.
